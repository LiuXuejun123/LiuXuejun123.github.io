UNIX 高级环境编程（APUE.h头文件配置）

2021-05-05 / 1 min read

材料：

环境：Centos 7

准备工作

官网下载素材：http://www.apuebook.com/code3e.html

选择自己的版本

下载后文件名为：src.3e.tar.gz

解压缩 ：

tar zxvf src.3e.tar.gz

头文件配置

在解压后的目录下 复制两个文件到 /usr/include 路径下

cp include/apue.h /usr/include/
cp lib/error.c /usr/include/

修改 /usr/include/apue.h

添加头文件‘’error.c‘’

准备环境

执行命令：

yum install https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm

 yum install libbsd libbsd-devel

编译

在apue.3e文件目录下执行

make

编译成功

Git 使用教程（1）初步命令

2021-07-01 / 2 min read

1.分布式版本控制器(Git)

1.安装方式见：https://git-scm.com/book/zh/v2/%E8%B5%B7%E6%AD%A5-%E5%AE%89%E8%A3%85-Git

2.文件管理：

文件git初始化

git init

查看当前文件夹下状态：

git status

对文件进行管理

git add 文件名

对所有文件管理

git add .

个人信息配置

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

生成版本

git commit -m '版本描述'

查看版本

git log

2.回滚

回滚实现，类似ctrl Z 重置为之前的版本

git reset --hard 版本号

前向回滚(可以看到全部的提交记录)

git reflog 
git reset --hard 版本号

3.分支

分支，将统一版本代码形成多个分支，同时进行开发。最后可以合并到一起

3.1 紧急修复线上bug的思路

分支的概念能够较好的实现对线上bug的修复。

主干线叫master(main)

分支可以自己定义

具体过程如下：

获取当前分支

git branch

创建分支：

git branch dev(分支名称)

切换到新的分支

git checkout dev(分支名称)

删除分支

git branch -d dev(分支名称)

合并分支：

git merge dev(分支名称)
